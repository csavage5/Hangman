# Hangman

This is a Python remake of Hangman.

## Files

**Hangman.py**

* Allows user to choose a difficulty
* Runs gameplay
* Displays rules
* Prompts player to replay the game

**dictionary-large.txt**

* Contains a list of words ordered alphabetically

**HMGraphics.py**

* Prints the graphics for Hangman
